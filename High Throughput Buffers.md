# Buffers
### Wash Buffer (2 L)
* 200 mL 1x PBS
* 1800 mL milliQ
* 4.08g/2L Imidizole (30 mM)
### BUGBUSTER Lysis Buffer
* 1x PBS
* 0.174 mg/mL (1mM PMSF)
* 1.5x Bug Buster
* 2mg/ml lysozyme (small scoop)
* 0.2mg/ml DNAse (small scoop)
### FREEZE/THAW Lysis Buffer
* 1x PBS
* 0.174 mg/mL (1mM PMSF)
* 2mg/ml lysozyme (small scoop)
* 0.2mg/ml DNAse (small scoop)
### Elution Buffer (500 mL)
* 50 mL 1x PBS
* 450 mL milliQ water
* 6.81 g/500mL Imidizole (200 mM)
