# Buffers
### Wash Buffer
* 1x PBS
* 30mM Imidizole
### BUGBUSTER Lysis Buffer
* 1x PBS
* 1mM PMSF
* 1.5x Bug Buster
* 2mg/ml lysozyme (small scoop)
* 0.2mg/ml DNAse (small scoop)
### FREEZE/THAW Lysis Buffer
* 1x PBS
* 1mM PMSF
* 2mg/ml lysozyme (small scoop)
* 0.2mg/ml DNAse (small scoop)
### Elution Buffer
* 1x PBS
* 250mM Imidizole
